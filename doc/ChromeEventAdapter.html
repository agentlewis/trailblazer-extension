Class: ChromeEventAdapter

ChromeEventAdapter

Listens to events emitted by Chrome's extension API and translates them into the format used by the extension. When an event is fired, an instance of this class will iterate over the registered listeners for that event, calling each with the translated event data.

Events

onClosedTab

Fired when a tab in any window is closed. Handlers receive a single argument ChromeEventAdapter.TabEvent which will look something like:

{
  type: "closed_tab",
  occurred: Date.now(),
  data: {
    tabId: tab.id
  }
}

onCreatedTab

Fired when a tab in any window is created. Handlers receive a single argument ChromeEventAdapter.TabEvent which will look something like:

{
  type: "created_tab",
  occurred: Date.now(),
  data: {
    tabId: tab.id,
    parentTabId: tab.openerTabId,
    url: tab.url,
    title: tab.title
  }
}
