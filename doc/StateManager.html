Class: StateManager

StateManager

Receives events through an instance of ChromeEventAdapter and manages the recording state associated with each tab. When a tab is recording, it uses events received to build the data structures necessary to represent a browsing trail.

This class should not have any browser-specific code in itâ€”that is restricted to concrete adapter implementations such as ChromeEventAdapter.
